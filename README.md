# Angel-s-Avoidance

* 天使となったプレイヤーが閉じ込められた檻の中でダークボールに当たらないように逃げ回るという<br>
コンセプトの「パソコンとスマホ（横画面）のどちらでも遊べる3Dアクションゲーム」です。

## Demo

* [ゲームプレイ動画](https://youtu.be/rJJZTU3cpYw)をご覧ください。<br>
（ファイルサイズの都合上、ゲームプレイ動画をYouTubeにアップロードしました。）
